Detailed Description

template<typename DataType>
class Array2d< DataType >

A 2D array class template

This array takes care of storing values in a grid. Behind the scenes it stores the data in a single vector which is needed for CUDA computations. When the data is accesed via the interface, the boundary type dictates the behaviour.
