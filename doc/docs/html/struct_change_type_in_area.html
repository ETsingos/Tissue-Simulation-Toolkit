Detailed Description

Request changing some of the particle's types

This specifies an area as a list of pixels, and requests a random selection of particles in this area of a given type to be changed to a different type.

This is used to implement the original adhesion generation algorithm, which is a bit funny biologically, but may be useful as a reference or something.
